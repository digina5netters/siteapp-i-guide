# siteapp-i-guide

## 项目简介

`siteapp-i-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或具体网站，仅作为静态页面的集合与索引。  
你可以在这里查看、引用或学习其中包含的 HTML 文件内容。

## 目录说明

```
├── pages/          # 存放所有归档的 HTML 页面文件
├── assets/         # 静态资源文件（如 CSS、JS、图片等）
├── index.html      # 页面索引/导航（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件均为独立页面，互不依赖。
- `assets/` 目录用于存放页面所需的共用资源。
- `index.html` 为可选的导航页，用于快速浏览所有归档页面。

## 页面归档说明

- 所有页面均为独立完整的 HTML 文件，可直接在浏览器中打开使用。
- 页面内容不涉及任何具体网址、域名、营销信息或诱导点击行为。
- 归档目的仅在于保存和展示 HTML 页面结构、样式与交互逻辑。
- 部分页面可能包含外部引用资源（如字体、图标库），请确保网络可访问。

## 维护说明

- 本仓库由维护者根据实际需要不定期更新。
- 欢迎通过 Issue 提交问题或建议，但请注意不讨论具体域名或网站内容。
- 如需提交新的归档页面，请将文件放入 `pages/` 目录，并更新 `index.html`（如有）。
- 请勿在仓库中加入任何违反平台政策的内容。

## 许可

本仓库内容仅供学习与参考。  
如有特殊引用需求，请参照具体文件的注释说明。
